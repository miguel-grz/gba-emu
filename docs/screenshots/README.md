# Screenshots

Drop the images referenced by the main [README](../../README.md) here:

| File | What to capture |
|------|-----------------|
| `library.png` | The **Library** view with a few games showing cover art (e.g. Pokémon FireRed, Emerald…). |
| `gameplay.png` | A game running in the **player** view (screen + controls dock). |
| `controls.png` | **Settings → Controls**, showing the remappable key/gamepad table. |

**How to capture on macOS:** run the app (`npm run tauri dev` or the browser),
frame the view, then press **⇧⌘4** and drag over the window (or **⇧⌘5** for more
options). Save each file with the exact name above into this folder.

Keep them reasonably sized (PNG, ~1400px wide is plenty) so the repo stays light.
